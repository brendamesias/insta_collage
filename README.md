# Reto 02: Insta Collage
## Instrucciones
+ En este reto debemos usar la API Drag and Drop de HTML5 para crear el collage arrastrando y soltando imágenes
+ Las imágenes son de libre elección
+ El mínimo de fotos para elegir deben de ser 6 y para el collage mínimo son 3
